# NBE DR Annex (Submission-Ready Summary)
**Annex: Disaster Recovery & Operational Continuity**

SelamNow enforces a multi-region disaster recovery architecture with strict GitOps governance.

## Key controls:
- Defined RTO/RPO per component
- Continuous replication monitoring
- Immutable infrastructure control
- No manual production mutation
- Quarterly failover testing
- Basel-aligned operational risk posture

## Evidence includes:
- Terraform state history
- ArgoCD sync logs
- Replication lag reports
- DR exercise documentation
- CI validation logs

System recovery is deterministic and regulator-auditable.
